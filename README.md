# Custom L298N Motor Driver PCB Module with 5V Regulator

Alhamdulillah, I have completed the design and layout of this custom, high-current 2-layer Motor Driver PCB. This module is specifically optimized for robotics, line-follower robotics applications where robust power management and noise reduction are critical.

## 🚀 Key Features & Specifications

* **H-Bridge Driver:** Utilizes the L298N multi-watt IC to control high-current DC motors.
* **Ground Plane (GND Shielding):** Implemented solid copper pours on  Bottom (B.Cu) layers. This creates an electronic shield that minimizes PWM switching noise and electromagnetic interference (EMI).
* **Advanced Via Stitching:** Strategically placed via arrays/stitching across the board to interconnect the top and bottom ground planes, ensuring a solid, continuous, and low-impedance current return path.
* **Onboard Power Supply:** Integrated an L7805 voltage regulator with decoupling capacitors to step down the motor voltage and provide a clean, stable +5V output for microcontrollers (like ESP32/Arduino) and sensors.
* **Flyback Protection:** Features SS34 SMD Schottky diodes (3A rated) to protect the driver IC from back-EMF inductive voltage spikes generated by DC motors.
* **Heavy-Duty Power Traces:** Power and motor output lines are designed with ultra-wide traces (up to 2.0 mm) to support high current without overheating or causing voltage drops.

## 🛠️ Design Tools & Workflow

* **EDA Software:** KiCad 9.0
* **Manufacturing Standards:** Fully compliant with 2-layer PCB design rules (DRC cleared), customized thermal reliefs for safe soldering, and optimal trace clearance settings.
